LearnKit SDK

Moving beyond Bookshelf

By building with LearnKit, developers can count on VitalSource to handle:

  • flexible content licensing terms
  • online and offline reading capability
  • content and metadata on-boarding
  • support for multiple content formats (EPUB 2, EPUB 3, PDF)
  • note / highlight / bookmark storage and cross-device syncing
  • lightning fast global content delivery
  • usage data collection and analytics feeds
  • up-to-date support for new browser and operating system releases
